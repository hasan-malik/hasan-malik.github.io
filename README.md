# hasan-malik.github.io

Personal portfolio website for Hasan Malik — built with React, Vite, Tailwind CSS, and Framer Motion.

## Stack

- **React 19** + **Vite**
- **Tailwind CSS** — custom Apple-inspired design system
- **Framer Motion** — animations and page transitions
- **GitHub Pages** — deployment

## Features

- Animated hero, project cards, experience timeline, and achievements
- Gallery page with lightbox and multi-photo posts
- Responsive, dark-mode-aware design
